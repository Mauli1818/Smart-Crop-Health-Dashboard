# 📖 Overview

Smart Crop Health Dashboard is an AI-powered web application that helps farmers monitor crop health using satellite imagery, weather information, and artificial intelligence.

The system analyzes crop conditions using vegetation indices and provides intelligent recommendations to improve crop productivity and reduce farming risks.
